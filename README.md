Calc-It
A collection of calculators I built to practice the JavaScript I learned in Angela Yu’s Web Development Course.

Why I built this
I didn't want to just follow a video; I wanted to see if I could build something from scratch. This project helped me practice the JS logic I just learned while making sure I didn't forget the HTML and CSS I studied earlier. I also used this as a way to try out new CSS ideas I’ve seen online.

What I practiced
As I moved from the first calculator to the last, I tried to make the code "cleaner" by looking up better ways to do things. Here are the main things I worked on:

HTML Tables & Inputs: I used proper table tags to show the results and used inputmode to make sure the right keyboard pops up on phones.

The DOM: I worked on the best way to connect my JavaScript to my HTML without the code getting tangled.

Math in JS: I used parseFloat() to handle numbers and Math.pow to do the heavy lifting for the harder formulas like the Credit Card repayment.

Money Formatting: I used toLocaleString so that the results actually look like currency (£) instead of just long strings of random decimals
